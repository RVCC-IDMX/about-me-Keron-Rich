## Project Description
This project is my final version of the About Me website that I have been building throughout the semester. The goal of this project was to learn the basics of web design and show that I can create a clean, organized, and working webpage using HTML and CSS. I also learned how to plan a layout, choose colors, add images, write content, and make the page look good on different screen sizes.

Throughout the project, I practiced several skills. I learned how to **wireframe** a page before building it, which helped me plan where everything should go. I added **hover effects**, a **navigation bar**, a **hero section**, a gallery, and a **contact form**. I also learned how to use **media** queries so the page works on phones, tablets, and larger screens. Another skill I practiced was adding a blog article and styling it with drop caps and a custom layout. I also created my own logo using my initials “KR” and added it as a favicon.

I made design choices based on what fit my personality and the color scheme I picked. I used bright colors to make the page feel fun and energetic. I also kept the layout centered and simple so everything is easy to read. Overall, this project shows the work I put in and the progress I made in web design.

## Color Scheme
These are the main colors I used in my design. I chose them because they stand out and match well together.

| Color Name | Hex Code | RGB Value | Usage |
|-----------|----------|-----------|--------|
| Light Blue | `#00BDE0` | 0, 189, 224 | Borders, accents, buttons |
| Gray‑Green | `#42614E` | 66, 97, 78 | Headings, blog styling |
| Light Green | `#28E06F` | 40, 224, 111 | Section backgrounds |
| Yellow | `#E0BF28` | 224, 191, 40 | Main page background |

I checked my colors using a contrast checker to make sure the text is readable.

## Design Choices
- I centered most sections to keep the layout simple and clean.
- I used rounded boxes for the hobbies section to make it stand out.
- I added a hero image at the top to introduce the page.
- I used drop caps in the blog article to make it look more like a real article.
- I added hover effects to the gallery to make the images feel more interactive.
- I created a hamburger menu so the navigation works on mobile screens.
- I added my own logo and favicon to make the site feel more personal.
- I used consistent borders and colors to keep the design matching across all sections.
